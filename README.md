# alke-wallet
Creación de billetera digital simulada utilizando HTML, CSS, JS, Bootstrap y JQuery.
</br>
La billetera inlcuye funcionalidades como un menú principal, depósitos, retiros, transferencias y una página con los últimos movimientos.
</br>
Además, se simula la creación de usuarios sin utilizar backend (con localStorage) y transacciones entre los mismos.

## Recursos gráficos
- Ícono de billetera (SVG) utilizado como favicon en la aplicación.

## Instrucciones de uso:
Debido a que los usuarios deben estar en el localStorage del navegador, favor crear al menos dos usuarios y simular transacciones entre ellos si se desea probar las funcionalidades.
